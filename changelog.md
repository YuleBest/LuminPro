- 新增: 活动级黑名单功能，可指定应用在前台时禁止模块提升亮度
- 重构: 配置文件由多个独立文本文件合并为单一 `config.json`
- 重构: Web UI 前端框架升级为 Vue 3 + Vite，组件化重构全部页面
- 新增: Web UI 界面缩放、日志导出、设置卡片折叠、header 自动收起等功能
- 优化: Web UI 页面切换动画、亮度进度条动画、底栏导航动画
- 修复: 日志关键字过滤不生效的问题
- 修复: 修复其他已知问题
